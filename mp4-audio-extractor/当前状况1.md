## 7. 启动方式

### 首次使用

```bash
# 1. 确认环境
rustc --version    # 应为 1.95.0
node --version     # 应为 24.x
ffmpeg -version    # 应为 8.0.1

# 2. 安装依赖
cd mp4-audio-extractor
npm install

# 3. (可选) 安装 whisper ASR
powershell -ExecutionPolicy Bypass -File tools/setup-whisper.ps1

# 4. 启动开发模式
npm run dev
```

### 构建生产版本

```bash
npm run build
# 产物位于 src-tauri/target/release/
```

## 8. 已知限制与注意事项

| 项目 | 说明 |
|------|------|
| **前端无打包器** | 不修改 `src/` 下文件的相对路径引用 |
| **JSON 文件不支持注释** | tauri.conf.json / package.json 是纯 JSON，修改后不要加注释 |
| **spectrogram 热力图** | 使用 `pow(val, 0.6)` 非线性映射，低能量区可见性正常 |
| **Canvas 先显示再渲染** | 必须先 `chartsArea.classList.remove("hidden")` 再调 `renderAllCharts`，否则 Canvas 尺寸为 0 |
| **whisper 可选** | 未安装时不报错，文字时序图表显示安装提示 |
| **零新 Rust crate** | 更新3 不修改 Cargo.toml，whisper 纯进程调用 |
| **临时 WAV 自动清理** | whisper 识别完成后立即删除 `_whisper_temp.wav` |
| **安全** | 无网络请求、无数据收集、无云 API 调用 |
